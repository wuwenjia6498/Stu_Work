# 任务目标
在现有的“老约翰学员习作展示海报”工具中，新增一款“简约留白”风格的海报模板（Style 2）。
用户可以在左侧表单顶部自由切换两款模板，右侧 Canvas 实时预览对应样式的海报。

# UI 与状态更新
1. **新增状态变量**：在组件顶部增加一个状态 `const [templateStyle, setTemplateStyle] = useState('classic')`，默认值为 `'classic'`（经典粉红），可选值为 `'minimalist'`（简约留白）。
2. **左侧表单 UI 增加切换器**：在左侧配置面板的**最顶部**，新增一个“模板样式”选择区。请使用 shadcn/ui 的 `RadioGroup` 或 `Tabs` 组件，提供两个选项：“经典粉红”（对应原版）和“简约留白”（对应新版）。

# Canvas 渲染逻辑改造 (核心)
原有的资源预加载、`wrapText`（多行文本计算）、动态 Y 轴高度累加的基础逻辑保持不变。
请将 `drawPoster`（或你的绘图主函数）内部的绘制流程按 `templateStyle` 进行 `if/else` 分支处理。

## 样式 2 ('minimalist' 简约留白版) 的具体绘制规范：
请严格按照以下从上到下的顺序和样式要求进行绘制：

1. **底层背景**：
   - 整个 Canvas 背景填充浅灰色（如 `#F5F5F5`）。
   - 在浅灰背景上，留出四周约 `20px` 的边距，绘制一个**占据绝大部分面积的白色圆角矩形**底板（填充 `#FFFFFF`，圆角如 `16px`）。后续所有内容都在这个白板内绘制。

2. **头部信息区 (左右分栏布局)**：
   - **右上角**：绘制预加载的老约翰 Logo 图片，靠右对齐。
   - **左上角**：绘制一个粉红色的圆角胶囊形底色块（如填充 `#FF7A84`），里面写上白色文字“学员习作”。
   - **左上角下方**：依次靠左（左对齐）绘制三行文本：“阅读馆：[内容]”、“学 员：[内容]”、“书 目：[内容]”。字体颜色使用深灰色 `#333333`，字号适中。

3. **主标题区**：
   - 提取自表单的 `mainTitle`。
   - 居中对齐，字体颜色使用粉红色 `#FF7A84`，使用较大字号并加粗（如 `bold 40px sans-serif`），需与正文拉开视觉层级。

4. **习作正文区**：
   - 提取自表单的 `content`。
   - 颜色改为深灰色（如 `#555555`），正常排版，行高适中，左对齐。

5. **图片区（复用原有逻辑）**：
   - 排版逻辑与 Style 1 完全一致：如果有两张图，左图正常显示，右侧手写稿通过 `clip()` 裁切成圆形。坐标根据 Y 轴自动下推。

6. **老师评语区**：
   - “老师评语：（署名）”需加粗，颜色 `#333333`。
   - 评语正文颜色 `#555555`，左对齐，正常换行排版。

7. **底部信息区**：
   - 逻辑与 Style 1 一致：左侧放二维码，右侧放两行联系方式。字体颜色 `#333333`。

# 代码要求
- 确保 `templateStyle` 切换时，Canvas 能够立即触发重绘。
- Y 轴总高度的计算逻辑，两套模板可能略有差异（主要是头部和边距不同），请根据不同的 `templateStyle` 分别计算准确的总高度，再设定 Canvas 的高度。
- 保证之前开发的 **“手写稿 OCR 识别”**、**“AI 生成评语”** 以及 **“内置图库选择”** 功能在切换模板后依然完美可用，不要破坏原有代码结构。
